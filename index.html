CI/CD  ·  Automation

Jenkins
Webhook Flow

Code ships itself. While the OG reads, the pipeline runs — commit to deploy, fully automated, zero drama.

⬆

Step 01

Git Push

Developer pushes to the feature branch. GitHub receives the commit and fires a webhook immediately.

github.com
🔗

Step 02

Webhook Trigger

GitHub sends a POST payload to Jenkins at jenkins.lab.local/github-webhook/ — Jenkins wakes up.

POST /github-webhook/
⚙

Step 03

Jenkins Pipeline

Jenkinsfile executes: checkout → build Docker image → run tests → tag with commit SHA.

Jenkinsfile
📦

Step 04

Push to Registry

Image tagged and pushed to the local Nexus registry at 192.168.2.128:8082.

192.168.2.128:8082
☸

Step 05

kubectl Rollout

Jenkins runs kubectl set image — Kubernetes performs a zero-downtime rolling update across 2 replicas.

Deployed ✓
✦

Step 06

Live at maximuslabog.lab.local

Ingress routes traffic. Health probes pass. The OG didn't lift a finger — it just happened.
